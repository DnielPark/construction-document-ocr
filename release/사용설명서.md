# CM 현장 문서 OCR 프로그램 사용설명서

## 📦 프로그램 개요
- **프로그램명**: CM 현장 문서 OCR 프로그램
- **버전**: 1.0.0
- **기능**: 현장 도면 이미지에서 측점 데이터 추출 → CSV 변환
- **주요 기술**: Claude Vision API (Haiku 4.5) 활용

## 🚀 설치 및 실행

### 1. 실행 파일 다운로드
- `release/CM문서인식기.exe` (Windows)
- 실행 파일을 원하는 폴더에 저장

### 2. 필수 준비사항
- **Claude API 키** (https://console.anthropic.com/)
- **이미지 파일들** (PNG, JPG, JPEG 형식)

## 🖥️ 프로그램 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│ 왼쪽 패널 (컨트롤)              │ 오른쪽 패널 (캡쳐)     │
│                                 │                        │
│ 1. API 키 입력                  │ - 캡쳐 저장 폴더 정보 │
│ 2. OCR 폴더 선택                │ - 썸네일 그리드       │
│ 3. OCR 실행 버튼                │ - Ctrl+클릭 다중 선택 │
│ 4. 출력 폴더 열기               │ - 삭제 기능           │
│ 5. 캡쳐 모드 버튼               │                        │
│ 6. 진행 상황 로그               │                        │
└─────────────────────────────────────────────────────────┘
```

## 📝 주요 기능 사용법

### 1. OCR 처리 (기본 기능)
1. **API 키 입력**: Claude API 키 입력 (보안: 메모리 임시 저장)
2. **폴더 선택**: 이미지가 있는 폴더 선택
3. **OCR 실행**: "OCR 실행" 버튼 클릭
4. **결과 확인**: `output/폴더명.csv` 파일 생성

### 2. 화면 캡쳐 모드 (실무용)
1. **저장 폴더 설정**: 오른쪽 상단 "폴더 변경" 버튼
2. **캡쳐 시작**: "📸 캡쳐 모드" 버튼 클릭
3. **영역 선택**: 드래그로 캡쳐할 영역 선택 (ESC: 취소)
4. **자동 저장**: `capture_001.png`, `capture_002.png`...
5. **반복**: 다시 버튼 클릭 → 새 캡쳐

### 3. 이미지 관리
- **단일 삭제**: 썸네일 우클릭 → "삭제"
- **다중 선택**: Ctrl+클릭 (파란색 배경 표시)
- **일괄 삭제**: "🗑️ 선택 삭제" 버튼

## 🔧 고급 기능

### 듀얼 모니터 지원
- 프로그램은 모니터 1, 캡쳐는 모니터 2 가능
- 전체 화면 오버레이로 모든 모니터 커버

### 출력 폴더
- `output/` 폴더에 CSV 파일 저장
- "📁 출력 폴더 열기" 버튼으로 바로 접근

## ⚠️ 주의사항

### 보안
- API 키는 프로그램 종료 시 자동 삭제
- 파일 저장 안 됨 (메모리 임시 저장)

### 파일 형식
- **입력**: PNG, JPG, JPEG
- **출력**: CSV (UTF-8 BOM)
- **캡쳐**: PNG

### 권장 사양
- Windows 10/11
- 8GB RAM 이상
- 인터넷 연결 (Claude API 호출)

## 🛠️ 문제 해결

### 1. API 키 오류
- Claude API 키 확인 (https://console.anthropic.com/)
- 인터넷 연결 확인

### 2. 이미지 처리 실패
- 이미지 파일명에 특수문자 없애기
- 이미지 해상도 확인 (너무 크지 않게)

### 3. 캡쳐 안 됨
- 저장 폴더 먼저 선택
- 관리자 권한으로 실행 (Windows)

### 4. CSV 파일 안 열림
- Excel에서 "데이터" → "텍스트/CSV"로 열기
- 인코딩: UTF-8 BOM 선택
