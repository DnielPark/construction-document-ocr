CM 현장 문서 OCR 프로그램 v1.0 - 사용 설명서
═══════════════════════════════════════════════════════════════

📌 프로그램 소개
하수관로 공사 수량표를 자동으로 인식하여 CSV 파일로 변환하는 프로그램입니다.
Claude Haiku 4.5 Vision API를 사용하여 이미지에서 표 데이터를 추출합니다.

───────────────────────────────────────────────────────────────

🔑 1단계: Claude API 키 발급 (최초 1회)

[1-1] Anthropic 콘솔 접속
https://console.anthropic.com

[1-2] 계정 생성 또는 로그인
- Google, GitHub 계정으로 간편 로그인 가능

[1-3] API 키 발급
1. 콘솔 로그인 후 "Get API Keys" 메뉴 선택
2. "Create Key" 버튼 클릭
3. 키 이름 입력 (예: "현장문서 OCR")
4. "Create Key" 클릭
5. 생성된 키 복사 (sk-ant-로 시작하는 문자열)
   ⚠️ 키는 한 번만 표시되므로 반드시 복사하세요!

[1-4] 비용 정보
- Claude Haiku 4.5: $1/1M input tokens
- 이미지 1장 ≈ 1,000 tokens = 약 $0.001 (1-2원)
- 월 100장 처리 시: 약 100-200원
- 월 1,000장 처리 시: 약 1,000-2,000원

💡 팁: 신규 가입 시 무료 크레딧 제공 (약 $5)

───────────────────────────────────────────────────────────────

📥 2단계: 프로그램 다운로드

[2-1] GitHub Release 페이지 접속
https://github.com/DnielPark/construction-document-ocr/releases

[2-2] 최신 버전 다운로드
- "CM문서인식기_v1.0.zip" 파일 다운로드

[2-3] 압축 해제
- 원하는 폴더에 압축 해제
- CM문서인식기.exe 확인

───────────────────────────────────────────────────────────────

🚀 3단계: 프로그램 사용

[3-1] 프로그램 실행
- CM문서인식기.exe 더블클릭

[3-2] API 키 입력
- Claude API 키 입력 (1단계에서 발급받은 키)
- ⚠️ 보안: API 키는 메모리에만 임시 저장
- 프로그램 종료 시 자동 삭제됨
- 파일에 저장되지 않음

[3-3] OCR 처리
① "폴더 선택" 버튼 클릭
② 수량표 이미지가 있는 폴더 선택
   (지원 형식: PNG, JPG, JPEG)
③ "OCR 실행" 버튼 클릭
④ 진행 상황 확인 (로그 창에 표시)
⑤ 처리 완료 후 "출력 폴더 열기" 클릭

[3-4] 결과 확인
- output/ 폴더에 CSV 파일 생성
- 파일명: 선택한폴더명.csv
- Excel에서 열어서 확인 가능

───────────────────────────────────────────────────────────────

📸 4단계: 화면 캡쳐 기능 (선택)

[4-1] 캡쳐 폴더 설정
- 오른쪽 "저장 폴더" 영역에서 "폴더 변경" 클릭
- 캡쳐 이미지를 저장할 폴더 선택
  (신규 폴더 생성 가능)

[4-2] 화면 캡쳐
① "📸 캡쳐 모드" 버튼 클릭
② 화면이 반투명 오버레이로 덮임
③ 마우스로 드래그하여 영역 선택
④ 마우스를 놓으면 자동 저장
⑤ 캡쳐한 이미지가 오른쪽 그리드에 표시됨

💡 듀얼 모니터 지원: 두 번째 모니터에서도 캡쳐 가능

[4-3] 이미지 삭제
- 우클릭 → "삭제" 선택
- Ctrl + 클릭으로 여러 개 선택 후 삭제 가능

[4-4] OCR 처리
- 캡쳐한 폴더를 "폴더 선택"에서 선택
- "OCR 실행" 클릭

───────────────────────────────────────────────────────────────

📊 5단계: 데이터 확인

[5-1] CSV 파일 구조
생성된 CSV 파일은 다음 13개 항목을 포함합니다:
- 측점
- 지반고
- 관저고
- 터파기 육상
- 터파기 수중
- 되메우기 관주위
- 되메우기 관상단
- 파취 및 복구 ASP
- 파취 및 복구 Con'c
- 파취 및 복구 보도블럭
- 모래부설
- 보조기층
- 동상방지층

[5-2] Excel에서 열기
- CSV 파일을 Excel로 드래그 앤 드롭
- 또는 Excel에서 "데이터 가져오기" 사용

❓ 문제 해결

[Q1] 프로그램이 실행되지 않아요
A: Windows 10 이상에서만 작동합니다.
   Visual C++ Redistributable이 필요할 수 있습니다.
   Microsoft 사이트에서 무료 다운로드 가능합니다.

[Q2] API 키 오류가 발생해요
A: - API 키를 정확히 입력했는지 확인하세요
   - sk-ant-로 시작하는지 확인하세요
   - Anthropic 콘솔에서 키가 활성화되어 있는지 확인하세요

[Q3] 인식률이 낮아요
A: - 이미지 해상도를 높여보세요 (300dpi 이상 권장)
   - 스캔 품질을 개선하세요 (명암 대비, 선명도)
   - 표가 명확하게 보이는지 확인하세요

[Q4] 듀얼 모니터에서 캡쳐가 안 돼요
A: - 프로그램을 최신 버전으로 업데이트하세요
   - 화면 배율 설정을 확인하세요

[Q5] 비용이 궁금해요
A: - Anthropic 콘솔의 Usage 메뉴에서 실시간 확인 가능
   - 이미지 1장당 평균 1-2원 예상
